Final Strategy & Architecture

API-Driven Platform

The entire solution is powered by Django Rest Framework (DRF), enabling secure, scalable, and future-ready integrations across greenhouse devices and user interfaces.

  • Clean separation between presentation and control logic
  • Supports external integrations and mobile clients
  • Optimized for performance and reliability

Panel App

Responsible for rendering dashboards and analytical views, covering both administrator and grower panels with insight-driven UI components.

  • Responsive templates for admin & user experiences
  • Data visualization and greenhouse analytics
  • Modular components ready for iteration

Core App

The automation backbone coordinating device connectivity, command execution, and real-time greenhouse telemetry.

  • Secure device-server communication channels
  • Event-driven command execution
  • Resilient data capture and monitoring

Reg_Log App

Authentication services ensuring controlled access via admin-managed registrations and JWT-based login mechanisms.

  • Admin-only onboarding workflow
  • JWT authentication for web & API clients
  • Security-first session and token policies
